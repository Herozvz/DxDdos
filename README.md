# Инструмент для асинхронного стресс-тестирования HTTP

![Python Version](https://img.shields.io/badge/python-3.7%2B-blue)
![Status](https://img.shields.io/badge/статус-в_разработке-orange)

Высокопроизводительный скрипт на базе `asyncio`, предназначенный для проверки устойчивости веб-серверов и сетевой инфраструктуры к высоким нагрузкам. Инструмент позволяет эмулировать тысячи одновременных подключений с минимальным потреблением ресурсов системы.

> **Перевел и оптимизировал: Herozvz**

## ⚡ Особенности

* **Асинхронная архитектура:** Использование неблокирующего ввода-вывода для управления огромным количеством соединений.
* **Динамическая подмена данных:** Рандомизация `User-Agent`, `Referer` и параметров запроса для обхода простых систем кэширования.
* **Поддержка различных методов:** Ротация методов `GET`, `POST`, `HEAD`, `PUT`, `OPTIONS` и `TRACE`.
* **Спуфинг заголовков:** Использование `X-Forwarded-For`, `X-Real-IP` и других заголовков для имитации распределенного трафика.
* **Поддержка SSL/TLS:** Работа с протоколами HTTP и HTTPS (включая режим игнорирования невалидных сертификатов).
* **Аналитика в реальном времени:** Встроенный счетчик запросов (RPS), активных соединений и ошибок прямо в консоли.

📊 Технические детали

Скрипт работает по принципу удержания открытых TCP-соединений (Keep-Alive) и агрессивной отправки HTTP-пакетов. В отличие от обычных тестеров, он не дожидается полной загрузки тела ответа от сервера, что позволяет максимально нагружать ресурсы сервера (CPU/RAM) при минимальной нагрузке на канал атакующего.
